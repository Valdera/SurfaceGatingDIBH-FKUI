# Untuk merekam patient
1. Klik tombol patient
2. masukkan id patient jika sudah ada patient, klik tombol create new patient untuk membuat patient
3. Jika sudah ada tampilan graph, pilih com yang terdapat arduino, tekan tombol start
4. Jika ingin merekam respirasi panjang klik push button kemudian jika sudah selesai klik lagi push button dan tekan save
5. Pastikan graph tidak berjalan ketika ingin berpindah ke menu lain (Jika masih berjalan klik tombol stop)
